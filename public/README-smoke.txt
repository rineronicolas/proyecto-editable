Coloca aquí tu archivo de humo en movimiento. El código buscará automáticamente en este orden:

1. `/public/smoke.mp4` (recomendado)
2. `/public/smoke.webm` (alternativa)
3. `/public/smoke.gif` (fallback si no hay video)

Recomendado:
- Formato: MP4 o WebM para mejor rendimiento y compresión; GIF también funciona pero es más pesado.

Después de añadir el archivo, reinicia el servidor de desarrollo:

```bash
npm run dev
```
